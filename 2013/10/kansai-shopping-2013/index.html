    。2013日本關西賞櫻去。購物一覽

    Oct 24, 2013
    2013京阪神奈
    日本 / 關西 / 櫻 / 購物 / 食 / kit kat / jagabee / pure軟糖 / アルフォート / 舞妓仙貝 / 西利漬物 / Royce / 大阪プチバナナ / 東京ばな奈 / 自由軒カレー / 札幌円山動物園ラーメン / 神戶プリン / くいだおれ太郎プリン / 神戸苺トリュフ

    去日本，免不了的就是敗家…
    不過相較於以往，現在Salu比較沒那麼熱衷於在日本購買美妝品了
    一是想要讓行李箱多點空間塞食物
    一是其實很多東西在台灣就買的到
    (當然，那種差價很大又是消耗品的生活必需品除外啦)
    所以今年關西這一趟，主要採買的以零食為主，以及免不了要買些孝敬親朋好友們的伴手禮~

    1/kit kat

    (圖片來源：http://d.nestle.jp/kitkat/limited/9416628.html)
    雀巢旗下的kit kat不用說了，這幾乎是必買品，台灣賣頗貴；
    可是好甜阿，但還是忍不住要帶回台灣~
    現在有啥口味可買可以看一下日本Kit Kat官網

    2/jagabee

    (圖片來源：http://www.calbee.co.jp/jagabee/product/)
    Salu購買的是這一款(Box Type)。
    jagabee號稱平價版的薯條三兄弟，
    近年在台灣一些日本零食店亦可看見販售，
    仔細比較的話在台灣可以看到比較不錯的價格就可以入手，
    日本買不見得比較便宜，且盒子體積又大會佔行李箱空間，所以建議還是在台灣購買就好
    (官網同圖片來源)

    3/伊右衛門茶飲

    (圖片來源：http://www.suntory.co.jp/softdrink/products/0000000003/FEA5G.html)

    伊右衛門茶飲是Salu的最愛，雖然日本很多茶飲也很不錯，但Salu就是覺得伊右衛門最好喝…
    在日本國喝不夠，還是想盡辦法帶回台灣XD
    日本便利商店147円左右(泛指500ml之size)，有時會特價(曾經買過105円、125円的價格)
    運氣好的話可以在super market遇到比便利商店便宜的價格
    Salu買過的最低價是松本清藥妝店的69円
    可能是出清品吧~
    今年綠、藍、黑三個口味都有喝到，藍色口味比較清淡，黑色相當濃醇，綠色則介於中間
    (官網同圖片來源)
    (2013/12更新：台灣現已有代理商進貨，不過價格高的要命，city super一罐要價新台票$50~60)

    4/pure軟糖

    這是Salu在台灣很喜歡吃的一款軟糖，台日價差10元內，
    但台灣的口味選擇性比較少，所以在藥妝店逛到還是會順手拿個幾包
    pure官網、軟糖各口味

    5/アルフォート巧克力

    反面長這樣

    有出極多種口味(Salu在日本有看到藍、綠、黃、咖啡、黑色包裝)
    Salu在台灣好像只有看到下面這種包裝？！

    (圖片來源：http://www.bourbon.co.jp/catalog/item?category=16&item=135)

    口味還算蠻不錯的，只不過一樣蠻甜的，Salu只好拿來配茶…
    (官網同圖片來源)

    6/舞妓仙貝
    舞妓仙貝是在錦市場購買到的，
    是咖哩口味的仙貝，只不過實在是太辣了，又有夠鹹，
    即使一次只咬個一片，還是很容易像包裝上的舞妓一樣，眼淚直流阿~

    7/西利漬物

    配飯吃很不錯
    唯一的缺點是保存期限有夠短，印象中才三、五天吧
    人口少的家庭會吃得相當辛苦
    京都很多地方都有店家，關西機場也有店舖，
    建議可以在京都的實體店面試吃決定品項後，在關西機場買(又可爭取一些保存期限)
    西利官網

    8/Royce

    這是買給Salu弟的，Salu好像僅有吃到一兩片…(來自姊姊的怨念)
    甜甜鹹鹹的巧克力洋芋片，讓人一片接一片，罪惡感?!吃完再考慮吧…XDD
    生巧克力也很好吃，但要挑一下口味…
    Royce Chocolate官網

    9/大阪プチバナナ
    大阪香蕉蛋糕~每盒八條入裝800円

    很貼心的設計….以及使用教學

    甜到爆!!

    大阪的香蕉蛋糕是瓢月堂的作品
    相較於東京香蕉蛋糕，甜度屬於死甜，容易膩，
    尤其是這回東京版&大阪版一同購入，高下立見~
    Salu還是比較推薦東京香蕉蛋糕囉~
    大阪プチバナナ官網

    10/東京ばな奈

    每盒八條入裝1000円~硬是比大阪香蕉蛋糕貴上200阿~~~

    內餡

    好吃的甜食….
    現在出了好多的版本
    最一般的版本在日本各大國際機場都買的到…
    東京ばな奈官網

    11/自由軒カレー(自由軒咖哩)
    大阪名物自由軒咖哩竟然也有出調理包？！
    在京都駅八条口附近的super market看到，馬上抓了放進購物籃

    橘色包裝是一般口味，黑色包裝是辣味咖哩~

    橘色包裝的一打開，比黑色包裝的多了一小袋裝秘傳特調醬汁~

    特調醬汁~很酸，但加入咖哩中很神奇的有一種香香的味道，且很下飯~
    咖哩味非常濃郁，但就是僅有咖哩醬汁，
    若想要有其他料品必須自己另外煮後，再與咖哩醬汁混和；
    黑咖哩倒是沒想像中的辣，
    跟市面上販售的咖哩塊煮出來的咖哩相比，這兩款我覺得的確是不太相同；
    橘色包裝口味蠻特別，下次若再前往關西，Salu還會購入~
    提供給無法前往自由軒享用大阪名物的遊客另一個選擇~

    12/札幌円山動物園ラーメン(札幌動物園泡麵)

    一款鹽味、一款醬油
    在路過京都哪個地下鐵站的小型super market見著的
    由藤原製麺所製作的札幌動物園拉麵
    比起一般拉麵店口味還來的清淡
    麵條屬細麵
    口味細膩
    Salu覺得吃起來很不錯，會再購入
    (竟然連wiki百科都有專門頁)

    13/神戶プリン(神戶布丁)
    來關西好像每個人都會入手神戶布丁；
    官網
    盒裝

    Salu挑的是這款，一盒兩入，印象中即要價525円

    吃起來…恩，沒甚麼特別的感覺，就是一般布丁咩(噗)
    不過果然還是比台灣的好吃(?!)

    14/くいだおれ太郎プリン(大阪食倒太郎布丁)
    袋子極可愛

    打開來被笑倒…

    食用方式教學

    食倒太郎，曾經在2008年倒閉、吹熄燈號，
    復出後的這款與堂島(就是大阪有名的堂島蛋糕捲啦！)合作所出的布丁不僅在外觀上做了特別設計，連布丁本體也與先前不同！
    一盒三入1050円，每顆布丁翻過來都附有一張食倒太郎的笑臉
    布丁中所附之兩包”調味料”算是與眾不同，
    微甜微苦的口感，在拆開長相搞笑的外型後，讓Salu更是為之驚艷
    Salu現今吃到的布丁名產中還沒吃過如此口味的布丁~
    長相可愛、口味獨特，算是很適合當伴手禮的一款名產
    食倒太郎也有出其他食品喔，可以見見官網
    大阪食倒太郎官網
    (註：[食倒]在日文中之意思為”吃到破產”，絕非字面上之吃到死or吃到倒在路邊喔！)

    15/神戸苺トリュフ(神戶莓巧克力)

    神戶有許多經典甜品
    這款是frantz銷售第一名的商品
    白巧克力中藏著一整顆的草莓
    雖然不是新鮮草莓、也享受不到大顆、juicy的感動，
    但在咬開巧克力後，微甜的白巧克力配合帶點微汁的淡酸草莓，口味真是特別阿~
    雖然白巧克力不是巧克力，但還是讓人忍不住想一口接一口吃下肚哩~~
    包裝非常精緻，相當適合拿來送禮
    該牌的其他商品(如布丁)好像也有許多人推薦，可以參考官網資訊囉~
    frantz官網

    關西機場算是很好買的機場
    名產相當豐富
    除了關西當地的名產(京都、奈良、神戶、大阪等)
    也可買到東京香蕉蛋糕，當然也包括台灣人很愛的北海道薯條三兄弟囉~(但印象中好像有購買盒數限制)
    Salu比較喜歡吃吃喝喝，故本篇主要以食品購買心得/資訊為主，提供給大家參考囉~
