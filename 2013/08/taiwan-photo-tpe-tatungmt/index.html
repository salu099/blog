    。201308 大屯山、台北百萬夜景

    Aug 28, 2013
    撮る
    台北市 / 大屯山 / 夜景 / 助航台 / 台灣北部 / 陽明山

    夏天是非常適合拍夜景的季節，
    只不過今年因為天氣實在太熱，
    台北動不動就飆上35度阿~
    以往總是能到處趴趴造的
    現在只要看到大太陽就躲起來啦XD
    一方面也是害怕中暑(今年已經中暑兩次)
    心情真是矛盾也~

    離題太遠~XD
    話說這天在爬山&拍照之間猶豫很久
    踢自己出門後一上捷運
    發現天空乾淨到不行~跟早上陰陰的老天爺判若兩人~
    二話不說馬上改行程
    到大屯山助航台拍夜景囉~

    大屯山：標高1093公尺
    助航台：騎車開車方便，免爬山走路，但請盡量減少開車，因這裡路小，不好停放喔！
    能見度：22K
    器材：D7000+T12-24+60(短焦較適合，因風大；長焦的話要找比較不會吹到風之處)
    (此處拍照條件受天氣影響相當大)

    1/

    視野非常棒，天氣好些甚至能看見林口

    2/

    遠眺101&麥帥

    3/

    天母球場、圓山飯店一一入眼簾

    4/

    聯絡大台北地區的各個橋
    標示了一下，才發現數量還真多ㄟ…..!!

    5/

    來個清晰無字版~

    6/

    不論是關渡平原、社子島，還有基隆河、淡水河…

    7/

    都看起來好清楚喔!

    8/

    連每天搭的捷運都看的到..
    還有劍潭附近的水管紅橋!
    看不膩、好樂阿!!

    9/

    這裡salu就不知道是甚麼了
    學識淺薄

    10/

    再看一次由密密麻麻的房子、溪河交織而成的台北城

    11/

    以及剛收割完的關渡平原，整齊的鋪蓋著棕色的拼圖

    12/

    松山機場

    13/

    麥帥特寫；
    還看的到饒河街夜市旁的彩虹橋!以及劍南山上的鄭成功祠~

    14/

    也見的到新兒童樂園

    15/

    16/

    今天站位不佳，拍不太到日落

    17/

    勉強擠到一旁，拍拍淡水小城

    18/

    美麗的夜景秀即將上映

    19/

    20/

    21/

    22/

    遠方可見點點的漁火~

    23/

    燈光顏色好豐富!!~
    今天的夕色帶著點霞光喔！

    24/

    25/

    八里五股一帶新蓋的一些建築很有特色
    點燈起來很漂亮

    26/

    超強大超美的大屯山夜色
    稱作臺北百萬夜景應該一點也不過分
    若也來個民眾票選，這裡應該很難不入選

    27/

    隨著時間越晚，天空也越乾淨，能見到的夜景範圍也越廣，
    最重要的是，風力也越強…(好冷)Orz
    (可惡，google的上傳品質好差..)

    最後就以這張蜿蜒的溪流夜色做結尾吧!
