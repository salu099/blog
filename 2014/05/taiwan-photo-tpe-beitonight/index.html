    。201401 北投、焚化爐觀景台夜色

    May 10, 2014
    撮る
    台北市 / 北投 / 焚化爐 / 社子大橋 / 關渡平原 / 夜景 / 台灣北部

    北投焚化爐觀景台，賞風景、吹冷氣、觀夜色

    相信不少人知道台北幾個公開可俯瞰臺北盆地的大樓觀景台
    101、站前三越等..
    今日Salu要介紹給大家的這個點位於北投關渡平原附近，
    是的，你沒看錯，[北投焚化爐]
    這個每天處理台北市垃圾的大本營，居然也有觀景台，甚至是景觀餐廳，
    雖然高度僅有116公尺，
    但觀看到的景色與101、站前三越截然不同，重點是觀景台無料，
    加上交通不若另外兩個景點那樣便利，前來的民眾並不多，不太需要卡位，Salu覺得在這看夜景很有品質呀XD

    前往拍照時間：2012~2014年間
    適合前往時間：全天
    使用工具：P300、D50、D7000、T124、60

    1/高高大大的煙囪內，有觀景台、景觀餐廳；另外北投焚化爐也有健身設備，但是不在此館

    2/電梯口貼著指標，按照指示按鍵上樓就可抵達

    3/左側大彎橋即是洲美快速道路，

    4/迷你可愛的小橘車們則是每天辛苦工作的垃圾車們

    5/下面則是關渡平原，此時因剛收成完畢，關渡平原除了樹林外整個是焦黃的土地

    6/天氣好時，可見101、站前三越；

    7/以及幾座往返台北新北的大橋

    8/俯瞰台北市的最後一塊淨土…..真希望這片田地不會消失不見，但聽聞這片土地早已被數個建商搶地完畢，等待可開發之日來臨..

    9/遠遠的方向竟出現一條短短的彩虹耶!~

    10/玻璃上貼有景觀說明

    11/反光真的非常嚴重，非常難避掉

    12/

    13/新通車的社子大橋，連結北投及社子島北部的交通

    14/忍不住再看一眼陽明山山腳下的這片台北市最後淨土

    15/春天時稻子未長成，會是綠油油的一片；改日再來訪拍個綠油油版本的關渡平原吧~

    16/然而淨土旁卻是數個資源回收站，令人不禁擔心作物的生長是否有疑慮

    17/

    18/華燈初上，台北燈火也一一亮起(又是避不掉的反光…)

    也喜歡這裡安安靜靜、不若其他很夯的夜景地點氣氛
    若想看看有別於水泥叢林的台北市風景的朋友，也可以來這走走喔^^
