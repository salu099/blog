    。201309 康樂山、遠眺大湖公園夜色

    Jul 18, 2014
    撮る
    台北市 / 親山步道 / 康樂山 / 夜景 / 大湖公園 / 台灣北部

    最近周末台北的午後天空很難有藍天，適合拍夜景的時間沒幾枚，只好清硬碟圖來打發時間-.-

    前往時間：2013/9
    使用器材：D7000+28-105+60
    適合廣角~標準焦段

    持續的找台北近郊的小山走走
    今天鍛鍊腳力的地點挑上了康樂山
    (胡說，根本只是為了拍這個景來的，才走個半小時哪來的鍛鍊腳力XD)
    前往山頂的步道有多條，Salu走的是黃石公廟站附近的登山口

    1/路邊不太明顯的步道入口

    2/路標寫著成功路5段350巷，兩旁都是人家，一點也不像是登山口阿…

    3/不打卡，改拍登山口標註，證明自己曾經來過

    4/一開始是平坦好走的水泥路

    5/後來換成泥土路

    6/鋪設的木頭不算不好走，但沿路皆為泥土路，加上有坡度，雨天過後的濕滑容易造成滑倒

    7/還來不及抵達山頂，眼見著太陽公公已準備下山，只好隨便挑了個路邊的點拍照

    8/今天西方有些雲層，太陽公公被雲層擋住

    9/隨著太陽西下，街燈也漸漸亮起

    10/天邊出現了一抹微紅

    11/可惜拍照時間與文湖線一直搭不上阿

    12/

    13/沒想到晚上看點燈的大湖公園還頗有氣氛

    14/今年夏天再來一趟吧！就這樣決定了

    15/

    16/拍來拍去，還是覺得水銀燈有氣氛…

    17/
