    小卷的胡言亂語

    隨心。隨興。隨喜。隨緣



    。201311 秋末出遊、宜蘭明池

    Feb 18, 2014
    撮る
    宜蘭 / 大同鄉 / 明池國家森林遊樂區 / 落羽松 / 黑天鵝 / 台灣東部

    陪著R兄消化年假
    來到這個早就想在秋天前來一探究竟的明池~
    這裡最有名的，不外乎秋天隨著季節轉換，而換上黃紅色調衣裳的落羽松
    另一個最大的賣點，就是池區內會聽音樂與樂師互動的黑天鵝…

    1/沿著員山鄉台七線往明池的方向前進，沿路滿眼綠意，很是舒服

    2/經過一路睡睡醒醒，林相終於與平地漸漸不同

    3/一根根佇立的枯木，是遭遇祝融後所殘存，在群山中點點綴著；偶然間飄過了一縷山嵐，好像增添了一股神祕感 :)

    4/明池遊樂區的住宿…看起來非常高級；印象中是非常的難訂…

    5/場區平面圖

    6/忽晴忽雨的天氣，讓天空掛上了一條彩虹!

    7/太陽忽然露臉，在陽光的襯托下，樹林的層次、搭配在陽光下的落羽松很美阿!~

    11/秋季時節裡，池邊經常見到許多攝影人取景

    12/環著湖，慢慢地走著

    13/可惜Salu手拙，拍不出美麗的倒影~

    18/遊樂區內的黑天鵝，只要聽到音樂聲就會以鳴叫聲呼應，並且與樂師互動..

    19/黑天鵝又悠悠地離去…(有關黑天鵝的故事，可以參考此網站，一個樂師與黑天鵝的故事)

    20/錦鯉悠游池中..

    21/被滿山的翠綠包圍，讓明池多了一股幽靜….

    26/據說是模仿京都龍安寺的造景設計…

    27/坐在禪亭發了一下呆…

    28/最後就以這張仿日庭園造景作結尾吧!

    明池是個幽靜的好地方
    建議至少還是來個兩天一夜遊，才不會像Salu一樣走馬看花
    除了秋季的落羽松，以及明池中的美麗黑天鵝，這裡夏天也是熱門的避暑聖地，聽說春天也有櫻花可賞呀
    只可惜今年來不及了，明年再來補考吧，一併把這回沒有拍到看到的美景裝進記憶卡 :)

    景點資訊

    明池國家森林遊樂區
    海拔1200m
    網站：http://www.yeze.com.tw/mingchih/homepage.htm
    交通：
    大眾運輸-馬告生態園區(明池、棲蘭)有神木群一日遊行程，由台北國父紀念館出發，詳情可參考該網站
    自行開車-
    宜蘭->宜蘭市內台七線直直走即可抵。(台七與台七甲分岔處起算至明池約20km)
    桃園->桃園大溪國小附近台七線直直走，過角板山、鷹山可抵。
