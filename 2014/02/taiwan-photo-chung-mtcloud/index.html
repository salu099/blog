    。201311 秋末出遊、遇上梨山雲海

    Feb 02, 2014
    撮る
    台中 / 雲海 / 梨山 / 参山國家風景區 / 台灣中部

    秋冬，雲海的季節
    每逢秋冬鋒面抵達前緣，以及鋒面遠離後，海拔稍高的山區就有機會可以觀賞到雲海
    Salu這回到梨山，剛好遇上鋒面抵達；
    本來以為沒希望了，因抵達當晚，梨山整個飆大雨；
    隔日一早，剛好在另一波雲雨抵達前的空檔，遇上雲海秀，
    剛好拍完~雨雲又至~只能說，天氣變化真是迅速阿，要拍照，真的要把握那瞬間，
    否則可是過了這個村，就沒這個店了呀~
    一起來欣賞雲海吧XDD

    相關資訊

    梨山交通
    宜蘭<->梨山、豐原<->梨山間有客運往返
    國光客運[宜蘭<->梨山]
    豐原客運[豐原<->梨山]

    住宿
    梨山有多間飯店、民宿可選擇，建議旺季(EX賞楓期間)提早訂房；
    Salu的照片多張是在所宿民宿新館所攝(1~14張)。
    民宿名稱有一字為某種鳥類。
    Salu住舊館，設備蠻不好，隔音也很差，Salu因行程決定時間倉促，故無法選擇，建議還蠻care居住品質者不用考慮舊館。
