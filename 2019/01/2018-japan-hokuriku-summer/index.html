    2018/7 說走就走的日本北陸小旅行-行程總覽

    Jan 24, 2019
    2018北陸
    日本 / 北陸 / 上高地 / 金澤 / 行程

    最近的台灣正處隆冬，那麼就來些熱呼呼的夏季小旅行回憶錄吧XD

    標題是說走就走，是的，秉持著友人熱愛的說走就走的精神，
    咱們看看機票然後制定行程+訂房+做些旅行事前準備，前後大概花一周左右的時間吧嗯XD
    當然一周能做的準備也是有限der，所以最後折衷的作法是四日小旅行，
    然後咧經過這回僅一周(而且還是出發前一周)的時間準備考驗後，傻露還是誠心認為，
    嗯這種說走就走小旅行還是待國內就好惹，出國玩實在太太太累啦噗哈XDD
    那麼以下就是這次的小旅行紀錄啦。

    2018/7北陸之旅

    • 去程12:05(TPE)-16:10(KMQ) (虎航)
    • 回程17:15(KMQ)-19:45(TPE) (虎航)
    • 四天三夜
    • 住宿 - Hotel MyStays(Kanazawa Castle)、旅館岐山、Daiwa Roynet Hotel Kanazawa
    • 吃喝飲食記錄
    • 小松機場、金澤往返交通-JR北陸本線+小松機場交通車

    day1

    • 起飛啦~台灣天氣很好，能見度超級好。下午4點多抵達小松，入境後搭巴士到JR小松站
    • JR小松->JR金澤
    • 晚餐麵屋大河
    • 飯後金澤站附近晃晃
    • 居酒屋笑笑
    • 宿Hotel MyStays(Kanazawa Castle)

    day2

    • 早餐gusto
    • 租車
    • 武家屋敷 野村家、武家屋敷通
    • 老舖紀念館
    • 午餐金沢まいもん寿司 本店
    • 下午開車前往奧飛驒
    • 晚餐旅館
    • 宿旅館岐山

    day3

    • 早餐旅館
    • 上高地
    • 雨晴海岸
    • 晚餐ゴーゴーカレー 金沢駅総本山
    • 宿Daiwa Roynet Hotel Kanazawa

    day4

    • 兼六園
    • 近江町市場
    • 早午餐じもの亭
    • 東茶屋街
    • 交通：使用北鐵巴士一日券
    • 搭機回台啦，JR金澤->JR小松->小松機場
