    南投。金龍山夜琉璃

    Jul 12, 2019
    撮る
    台灣中部 / 夜景 / 日出 / 金龍山 / 琉璃夜景 / 雲海

    久違的文章來了，先離題日本金澤，來份台灣的美景。
    人生走這麼一遭，能親眼見到這樣的美景，值得了。
    老天爺恩賜的禮物，我就收下了。謝謝老天爺。

    今年好忙，加班根本稀鬆平常。這陣子總算比較有時間能整理硬碟的照片，寫寫心心念念想分享的照片。
    (被強迫跟著看的大家，辛苦啦XD)

    今年春天時趁著空閒時間，走訪南投。原先沒有規劃來到這個點，當然更別提甚麼能拍到夜琉璃這檔事啦。

    到訪南投的日子，前幾日雨下下停停，在海拔2000公尺的山上冷到一個不行，令人難忘。
    臨時起意，在旅程的最後一日，決定前往金龍山碰碰運氣。
    也許是心誠則靈(嘴上已叨唸了好些年，想拍夜琉璃)，
    又也許是老天爺的悲憫(至少十次的抓龜經驗)，
    當然我想最重要的是天候條件配合，讓我能夠一圓拍攝夜琉璃的夢，這個看似遙不可及的夢，卻在這樣臨時起意的時候讓我夢想成真XD

    當天&前一日南投的天氣狀態：

    • 拍攝季節為春天(3月)
    • 拍攝前一日南投午後有雨，但下雨時間似乎不長，傍晚前雨即停。
      拍攝前一日傍晚前去虎頭山撲空(雲海高度不夠低，白牆一片不斷上升)
    • 拍攝當日清晨在前往金龍山路上觀察到幾個特點(傻露住日月潭附近)：
      1. 空氣中水氣重(該不會是因為這次拍照，身體吸飽飽的溼氣所以導致現在鼻子過敏吧，好吧，相機沒事比較重要XDD)
      2. 平地有霧，但抬頭可見星星，天空清亮乾淨，路燈照下來可以看到白色的水氣、霧氣

    幾項特點似乎與網路上網友分享的拍攝經驗蠻符合的，至於一些常見的分享，如看逆溫層圖之類的資訊，我想應該僅能當作參考用，畢竟並非拍攝當時的數值。
    到目前為止，我認為拍攝夜琉璃這種題材，是屬相當吃人品的題材，
    或許網路上不少人提供一些判斷天候的方式，
    那除了是”稍微提高可以拍到出景的機率”(但不保證一定會成功，充其量只能說機率較平常高一些)，也只是一個通則，不見得適用所有易出現雲海的地點。
    不過或許日後的科技再進步些，判斷出景的機率或許能更提高也不一定(好期待機器學習在這方面未來的應用啊XD)。

    金龍山，有名的拍雲海地點，也是台灣不少愛好風景攝影者琅琅上口的”北格頭、中五城、南二寮”其中的五城。
    當然也有不少其他適合拍攝雲海的地點，不過以上這幾個地點算是老牌&大家口耳相傳的有名雲海拍攝地點(缺點就是人擠人啊…)。

    好啦，趕緊結束落落長的一堆廢話，分享這次難忘的經驗與回憶吧XD。

    2/

    這天清晨沿著山路一路往上，一到觀景台，金龍山雲海大戲映入眼簾。
    那日夜裡，和我們一樣瘋瘋的、不睡覺的傻子守在這裡看著老天賜給我們的美景(雖然才兩三隻小貓^^||)。

    3/

    好感動、內心澎湃不已；這就是我期待已久的夜琉璃嗎，它現在就在我眼前，肉眼就可見

    4/

    相機濕了，擦乾繼續拍，沒一會水氣又上來；
    相機濕了又乾、乾了又濕，拍照總是很忙碌，要按快門要顧濕度，還要按手機的計時，不懂為何有的大師級的老師們還能泡茶聊天？
    (謎之音：這不是很明顯了嗎，防水措施+快門線預備備…)

    5/

    隨著日出時間接近，天空增添了一抹淡淡的橘色。看來今日天氣會很不錯呢。
    (謎之音：都要收假回家了才給我好天氣，哼，森77><)

    6/

    看著天空越來越亮，而山巒間的雲海又與前半小時不同了

    7/

    天將亮，雲海下的燈火也漸漸歇熄，換太陽接手照亮大地

    8/

    那些五顏六色的燈光一一離場，也宣告短暫的夜琉璃正式結束

    9/

    留下尚未飄散的雲海。再不出幾個小時，它們的生命也將結束。

    10/

    在雲霧飄渺的山間是甚麼感覺呢？印象中那是伸手不見五指，白茫茫的難以辨別自己身處方位。
    而現在在高處看著雲海，感受又有所不同，看著那隱隱約約的小房子、檳榔樹林，恍然明白為何許多人愛以這類題材入畫(或許就像攝影師喜歡把美景裝進記憶卡內一樣道理吧..)

    11/

    亮著跑馬燈的孔明廟，剛剛還有路過呢，明明才在山腳下而已，開車不到半小時的路程就已經離我好遠好遠。

    南投魚池鄉金龍山觀景臺

    後記

    雖然不是第一次來了，但第二次來就幸運地看到傳說中的美麗金龍山琉璃(第一次來遇到超級白牆，連個日出都沒影)，覺得自己真是幸運。
    寫著文，看著照片，又想起當天的情景：
    滿天星斗、高掛樹梢的月娘，以及掉根針在地上都能清楚聽見聲音的寧靜夜晚中，上演的雲海大戲。
    (當然更難忘的是器材使不上力時的慌亂XD)

    那一夜，是傻露這輩子第一次與琉璃光夜景相遇。
    大自然的奧妙難以一語道盡。走這麼一遭，人生真的值得了。嗯，人生原來還是有值得期待的事…(真心希望還能有第二次、第三次的夜琉璃…會不會太貪心？！)。

    註：拍照當時，手中能控制快門的遙控器不聽使喚不願上工，返回山腳下卻又恢復正常，
    這離奇的事件也是傻露這輩子第一次遭遇..
    究竟是老天爺想藉此勸我買新工具，還是肩包上的大悲咒在提醒我想要有新夥伴陪伴呢？
