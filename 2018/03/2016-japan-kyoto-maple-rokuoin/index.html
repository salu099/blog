    2016/12 日本京都小旅行 - 鹿王院

    Mar 26, 2018
    2016京都
    日本 / 京都 / 關西 / 秋 / 楓 / 鹿王院

    經過一個下午+晚上的風雨，隔日起床竟然出了個大太陽。

    今天前往鹿王院，
    鹿王院位在嵐電鹿王院站附近，算是京都市內比較晚楓紅的點，
    話說嵐山附近有許多點可以走走，好似怎麼也逛不夠，
    尤其是嵐電~，幾乎每一站都有得逛阿，
    傻露來了京都這麼多次，還是很喜歡安排到嵐山附近走走~。

    2/電車站下車後，在幽靜的巷弄內散步沒多久即可抵，門口已經有株紅葉在等著造訪啦

    3/一進門，就是這條長長的石頭路，搭上火紅的楓葉，以及藍天，真是漂亮~

    4/紅通通的楓葉，終於滿足我這回來京都看紅葉的願望了，嗚嗚..
    (說的好像很可憐，不過真的蠻慘的XD)

    12/除了落葉地毯，抬頭也是紅黃交織

    想不到鹿王院小小一枚讓人驚艷，讓傻露在這按了不少快門阿(狀態顯示為已滿足)
    喜愛幽靜的賞楓景點者，推薦可以前來鹿王院瞧瞧喔。
