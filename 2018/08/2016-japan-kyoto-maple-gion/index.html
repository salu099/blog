    2016/12 日本京都小旅行 - 京都祇園散策

    Aug 30, 2018
    2016京都
    日本 / 京都 / 關西 / 秋 / 楓 / 本能寺 / 三条 / 木屋町通 / 祇園 / 石塀小路 / 八坂神社 / 円山公園

    旅日終章，最後一日的行程來到祇園附近走逛

    雖然來了祇園不下數次，但實在太喜歡這裡的街道了，於是在最後一日還是決定來這附近走走。
    今日走的路線，從飯店=>本能寺=>三条=>木屋町通=>祇園=>石塀小路=>円山公園=>八坂神社。

    2/飯店隔壁的本能寺

    3/

    4/

    5/三条街道

    6/木屋町伊東豐雄的清水模作品，星巴克

    7/日本映畫(電影)發祥地。看起來好像是一所小學?!

    8/蕭條的木屋町

    9/京都的七味粉很有名；經過味幸，停下來買了罐抹茶、柚香七味粉(但都沒很喜歡就是了XD)

    10/

    11/新立的告示牌，看來京都人對觀光客的怨念好像越來越深了阿

    12/早上還沒甚麼人潮，店家也都還沒開店

    13/要知道能在京都熱鬧的一級景點區裡享受寧靜感很不容易啊

    14/

    經過石塀小路，可惜立著禁止拍照的告示，只好略過了

    15/不記得是哪間院廟的後門了

    16/很有風味的羊腸小徑，黃昏時應該很有氣氛吧

    17/小徑的另一側通往高台寺

    18/有名的甜品店都路里。印象中旅遊書上都是介紹這間餒

    19/

    20/

    21/時間不太夠，不然想進高台寺看看..

    22/

    23/青蓮院(這麼綠，害我以為自己是夏天來der)

    24/円山公園

    25/有名的賞櫻名所，秋日雖有些蕭條但也不少遊客

    26/枯枝的那株就是賞櫻時節最有人氣的櫻花樹了

    27/

    28/八坂神社

    29/很喜歡這裡的氛圍

    30/結果越往下走越熱鬧，還有攤販聚集於此

    31/say bye-bye啦。神社下回見~。

    祇園、石塀小路這一帶的建築很有味道，氛圍很棒，
    但這兒還是有不少民宅，建議靜靜的遊逛，不要大聲喧嘩；
    行程上也可以搭配五年坂、清水寺，
    雖然這兒是觀光熱區，但很值得花上一整天的時間細細品味這一區域的風采。

    題外話，京都五社剩下城南宮、松尾大社還沒去過了
    不過我沒有蒐集五社之印阿..我也想要四神色紙啦..(跺腳)
