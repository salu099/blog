    2016/12 日本京都小旅行 - 京都吃喝飲食紀錄

    Sep 07, 2018
    2016京都
    日本 / 京都 / 關西 / 食 / やよい軒 / 長次郎迴轉壽司 / 茶寮小森 / なか卯 / 豆雅傳 / 中村藤吉 / 松屋 / 丸龜製麵 / 壹錢洋食 / 洛匠

    簡單記錄一下本次旅行食記。

    彌生軒(やよい軒)

    2/壽喜燒定食(890隻羊)

    3/目玉燒定食(460隻羊)

    4/洋風漢堡排定食(760隻羊)

    在飯店附近就有一間彌生軒。
    這間連鎖的定食料理店傻露算常來吃，價格不貴，餐點品質也算可以，白飯還可免費無限量供應，cp頗高，有些店家甚至營業24小時，自助的旅人吃飯時間往往與一般人不同，這時候這種店家顯得好便利啊。
    台灣也有彌生軒，但在台灣不是24小時營業，走的路線好像算是比較高檔一些，價位偏中上。
    日本的彌生軒點餐採用食券販賣機，台灣則是每桌有平板點餐。

    長次郎迴轉壽司

    5/

    6/抹茶粉

    7/除了拿取軌道上的壽司，也可利用平板點餐

    8/

    9/語系除了日語，還有提供簡體中文，握壽司還能選擇是否加wasabi好貼心

    10/

    11/

    12/沒有醋飯&魚肉分離的狀況，有點感人

    13/種類蠻多的

    長次郎在地下一樓，用餐時間通常需要排隊。
    傻露挑下午時候前來所以沒啥人，但也因此在軌道上見不到甚麼壽司在跑，這點感覺有點用心說。
    點了一些握壽司，覺得吃起來鮮度還算不錯，也沒有魚肉很薄的狀況，不過似乎沒甚麼特別的魚種。
    抹茶粉的優點就是可以自己掌握濃淡，價格比平價的迴轉壽司稍高一些但還能接受。
    整體而言中規中矩，如果能不排隊，基本上還會想回訪，但甚麼時候還會回京都呢(咦)
    用餐完畢步出店家已經5點，此時已經開始有人在排隊了(驚)。
    冬天外面還蠻冷的，在外排隊等吃冷冷的握壽司，想著想著覺得自己胃好像都要痛起來了><。

    本次消費2人總價2,232隻羊，大概6分飽的狀態。

    茶寮小森

    14/

    15/

    16/

    17/

    18/菜單是卷軸，貌似完全手寫的，價位稍高

    19/蕨餅(旁邊的是黑糖蜜)(1100隻羊)

    20/聖代+熱抹茶套餐(=1450+400=1850隻羊)

    21/總計2950隻羊(比吃正餐還貴=.=)

    22/庭園

    小森需要脫鞋入內，茶寮整體感覺很舒服，還有庭園可愜意的邊賞景邊吃甜品(但這位置要用搶的XD)，無用餐時間的限制。
    本來傻露是要去另一家店的，但無奈該店家排著長長一串人龍，回程看到小森不需排隊，就入內了，
    點的甜品都很喜歡，蕨餅還蠻軟的、聖代的抹茶會苦喔，裡面又有不少的料，第一次吃這種聖代，一直有驚喜感出現。
    吃一口甜品，配一口微苦的熱抹茶，這種感覺太奢侈了，真不敢想像我也能這樣在茶寮享受。
    也許該感謝老天爺，若不是那家名店排隊的人潮，我也不會遇見小森吧，雖然與京都第一名的蕨餅無緣就是了..。
    唯一感覺痛苦的便是榻榻米了，實在無法像日本人那樣跪坐，所以怎麼坐都不舒適啊。

    なか卯

    23/朝食-牛小缽目玉燒(350隻羊)

    略顯普通，這個分量感覺男生一定吃不飽

    豆雅傳

    24/menu

    25/湯豆腐套餐(1550隻羊)

    26/豆腐唐揚丼(1200隻羊)

    27/

    28/田樂燒(520隻羊)

    京都豆腐相當有名，正宗的湯豆腐料理要價都是3000起跳，找資料時看到豆雅傳算是平價，就前來嘗嘗。
    豆雅傳用餐處位於二樓，一樓販售一些豆腐製品，扛棒又小小一枚，所以好容易錯過阿；
    這天來到此大約傍晚五、六點多，稍微等候一下就有座位了，
    座位大約20初不算多，感覺用餐時間如果不早點來會等很久。
    湯豆腐套餐略顯平淡，吃慣重口味的人可能不習慣；
    豆腐唐揚丼的白飯(圖中左下那碗)是泡在湯裡的，吃起來非常好吃，
    豆渣(圖內上中那碟小菜)吃起來不太習慣，但想想豆子拿來做豆腐、豆漿後還能將豆渣入菜而不浪費，也就乖乖吃下肚了。
    田樂燒其實就是各種口味的豆腐拿來沾不同的醬汁食用，嗯可能很多人覺得很新鮮，但口味上傻露不太能適應，只能說嘗過口味就好。
    平價又能吃到京都的名物豆腐，豆雅傳的c/p蠻高，傻露還蠻推薦的。

    中村藤吉

    29/450隻羊

    京都宇治抹茶相當有名，中村藤吉也是宇治相當有名的抹茶店家，
    沒時間去宇治，京都站就有中村藤吉，
    來隻好吃的抹茶冰淇淋，彷彿又想起2007年第一次在宇治中村藤吉吃到抹茶冰時的感動。
    但，漲價了..。(漲了不少阿)

    松屋

    30/狐狸烏龍麵(390隻羊)+生菜(110隻羊)

    31/エッグ定食(400隻羊)

    松屋也是許多旅人喜愛造訪的店家，傻露喜歡它更勝なか卯..。不過友人說松屋牛肉自從不是使用國產和牛後就沒那麼好吃了，傻露沒得比較不知道是不是真是如此?!
    也是平價又24小時的便宜好吃店家，愛店一枚。

    丸龜製麵

    31/ぶっかけ（大）(390隻羊)(濃湯烏龍麵)

    32/とろろ醬油(並)(350隻羊)(山藥醬油烏龍麵)

    還點了”イカ天”(110隻羊)(炸花枝)、”野菜かき揚げ”(130隻羊)(炸野菜)，
    天婦羅屑隨你加，價格又平價，隨便亂點一人通常一千隻羊有找，也是許多台灣旅人的愛店啦。
    唯一的缺點是通常營業時間不會到太晚，開門時間也接近中午，所以行程時間常delay的人可能不太適合。(就像傻露，噗)。

    壹錢洋食

    33/

    34/

    35/

    36/

    37/650隻羊

    38/

    多年前在書上曾看到這家貌似厲害的店家，一直覺得好像很有趣。
    這回某天晚上在飯店附近走逛看到一錢，好奇心驅使下進店瞧瞧。
    門口的擺設非常有趣，店內牆上也掛/放滿特別的蒐集
    (原子小金剛、迷你盆栽的擺飾品，還有..嗯，很難說，傻露是女性不好開口..)
    最特別的是會有假人在你身邊陪你用餐，一個人也不會覺得怪怪的(?!)
    店內除了飲料外只有販售一項商品，類似大阪燒的東西，好的，那麼就點來吃吃看吧，
    乍看之下很像蛋包(蔥+一些料)，內餡就如圖38那樣(也可以上官網看)，另外還有一小段黑輪、雞蛋、肉。
    我只能說實在非常鹹，鹹到我也吃不出來到底內餡是甚麼?不過目測看來是蒟蒻??嗯我想我可能這輩子吃這一次就夠了，沒有很合我的胃口。
