    2016/12 日本京都小旅行-秋楓寶筐院

    Dec 30, 2017
    2016京都
    日本 / 京都 / 關西 / 秋 / 楓 / 寶筐院 / 嵐山

    這是第一次搭乘廉航+紅眼班機(香草的第一次，就給飛關空了)，
    原以為夜晚的飛機會很少人呢，結果幾乎客滿，
    香草座椅舒適度還算ok，座椅普通大小，整個航程3小時多還算可以接受，
    運氣很好沒有遇到座位附近乘客點食(搭機前怕被食物味道攻擊肚子整個餓起來XD)，
    但背後坐了個會踢椅子的屁孩(天啊)
    機師技術還不錯整趟飛下來感覺蠻穩的(拇指)

    抵達關西空港的時間已是半夜2點多，
    雖然看起來這時間只有香草這班航班，但因人數不少，入關檢查還是稍微排隊了一下，
    領完行李看看手錶也才接近3點，入關手續只花半小時算是快速~
    在機場內公共區域找排長椅睡一下，雖燈光非常亮，但能免費平躺休息已感到幸運
    隔日一早開始有旅客走動，聲音讓人很難不清醒，趕緊起身整理行李前往市區吧。

    早安，關空機場!

    時間6點左右，拎了小麥早餐，到JR綠色窗口換了Haruga來回的票券
    看著天空逐漸亮起，終於有實實在在此刻身在日本的感覺。

    嵐山

    在京都車站買了京都地鐵+嵐電一日券後，前往飯店先寄放行李，之後就出發前往嵐山了!

    嵐電西院站(這是我第一次來到這個站)

    嵐電嵐山站巧遇kumamon塗裝車廂，這是因應熊本地震的應援活動

    沿路就有楓葉可賞

    落葉也美

    天氣很好，天很藍，可惜紅葉掉了不少

    寶筐院

    寶筐院歷史解說牌

    一般相機可入內，專業相機/腳架不可(傻露拿中階dslr沒問題)

    販售的明信片

    落葉紅毯，可惜多已乾枯

    多種顏色在太陽光下更有層次，好美

    很喜歡那彎曲的線條配上紅黃色調

    雖然地上的紅葉比樹上的多..(泣)

    但還是有點感動

    在堂內發呆看著這一幕，多希望時間能一直停留在這

    坐在小小的庭園前發了一下呆

    雖然相較其他賞楓名所，寶筐院迷你了些


    但與大景點比起來，我還比較喜歡小院廟的清閒

    抬頭看看佈滿天空的楓葉

    最後就以這張寶筐院內小徑做結尾吧

    景點資訊

    寶筐院
    網站
    大人：500 円／小中学生：200 円
    營業時間：9:00～16:00
    注意事項：腳架不可、專業相機不可(數位相機可)入內
    紅葉見頃時間：11月底~12月初，官網有歷年照片可看，想要安排此點者可以參考寶筐院最新情報
