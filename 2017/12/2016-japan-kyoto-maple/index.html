    2016/12日本京都小旅行

    Dec 16, 2017
    2016京都
    日本 / 關西 / 京都 / 楓 / 光明寺 / 鹿王院 / 北野天滿宮 / 小森 / 洛匠 / 下鴨神社 / 光明院 / 本能寺 / 寶筐院 / 寶嚴院 / 行程

    這是拖了一年的遊記XDD

    出發前兩周臨時起意，決定到京都賞楓，不過還真是賭錯了，沒有看到最美的楓葉XD
    不是楓葉尚未完全變紅，就是秋風掃落葉(歪腰)。
    不過能出遊還是挺開心的~。
    雖然關西已經去過多次，不過沒啥時間準備
    所以和友人決定行程當日視楓況/天氣/體力/心情再決定當日的行程是賞楓/室內/購物
    鎖定目標在京都市區內、慢遊，整個就是走隨興的風格這樣XD

    2016年京都楓況聽說沒有很好
    主要是出發前幾日關西突然天氣變得不是很穩定，下雨打落了楓葉
    加上有些名所的楓葉變色較慢(好像是因為今年秋天較溫暖，溫差沒那麼大) 所以賞楓就變成賞黃葉+落葉了這樣(哀傷)
    不過落葉也有落葉的美啦(但新鮮的落葉比較美啊)
    那麼以下就紀錄一下這次行程啦

    2016/12京都之旅

    • 去程22:45(TPE)-02:15(KIX) (香草)
    • 回程20:35(KIX)-22:55(TPE) (虎航)
    • 五天五夜
    • 住宿 - 格蘭小姐
    • 關西機場往返交通 - 使用haruga+icoca方案(來回)

    day1

    • 凌晨2點多抵達KIX，在機場簡單打盹
    • 機場買了小麥早餐、搭乘はるか直達京都，飯店寄放行李
    • 嵐山寶筐院
    • 龍安寺
    • 嵐山寶嚴院夜楓
    • 早餐 - 小麥/午餐 - 路邊冰淇淋+可樂餅+飯糰/晚餐 - 彌生軒

    day2

    • 本能寺
    • 光明寺
    • 午後變天，早早收工去吃壽司逛街採買、祇園小森品嘗抹茶聖代
    • 早餐 - 進進堂麵包/午餐 - 長次郎壽司/晚餐 - 小森(因為吃完就飽了所以..冏>)

    day3

    • 鹿王院
    • 北野天滿宮
    • 早餐 - なか卯/晚餐 - 豆雅傳

    day4

    • 三條散步
    • 光明院
    • 下鴨神社
    • 早餐 - 松屋/晚餐 - 丸龜製麵/宵夜 - 壹錢洋食

    day5

    • 三條/木屋町通/祇園/石坪小路
    • 洛匠歇腿吃抹茶聖代
    • 八坂神社、圓山公園
    • 關西空港吃晚餐，掰掰日本下次見
    • 早餐 - 彌生軒/下午茶 - 洛匠/晚餐 - 關西空港拉麵
