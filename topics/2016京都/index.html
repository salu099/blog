    2016京都
    日本 / 京都 / 關西 / 秋 / 楓 / 光明寺

    還記得第一次出國，那也是我第一次自助，就來到京都光明寺。
    雖然第一次出國自助就來過這兒了，
    但這次前來京都還是來看看….因為好多景點的楓葉不是掉光就是變枯葉惹(泣)
    既然楓葉情報網站上說光明寺楓況還不錯，於是就打算來瞧瞧，
    結果光明寺的楓葉沒有預期中的紅阿(哭)，只好當作舊地重遊(么么)
    anyway，還是紀錄一下光明寺的(不夠紅的)紅葉囉。

    2016/12 日本京都小旅行-秋楓寶筐院

    Dec 30, 2017
    2016京都
    日本 / 京都 / 關西 / 秋 / 楓 / 寶筐院 / 嵐山

    這是第一次搭乘廉航+紅眼班機(香草的第一次，就給飛關空了)，
    原以為夜晚的飛機會很少人呢，結果幾乎客滿，
    香草座椅舒適度還算ok，座椅普通大小，整個航程3小時多還算可以接受，
    運氣很好沒有遇到座位附近乘客點食(搭機前怕被食物味道攻擊肚子整個餓起來XD)，
    但背後坐了個會踢椅子的屁孩(天啊)
    機師技術還不錯整趟飛下來感覺蠻穩的(拇指)

    2016/12日本京都小旅行-行程總覽

    Dec 16, 2017
    2016京都
    日本 / 關西 / 京都 / 楓 / 行程

    這是拖了一年的遊記XDD

    出發前兩周臨時起意，決定到京都賞楓，不過還真是賭錯了，沒有看到最美的楓葉XD
    不是楓葉尚未完全變紅，就是秋風掃落葉(歪腰)。
    不過能出遊還是挺開心的~。
    雖然關西已經去過多次，不過沒啥時間準備
    所以和友人決定行程當日視楓況/天氣/體力/心情再決定當日的行程是賞楓/室內/購物
    鎖定目標在京都市區內、慢遊，整個就是走隨興的風格這樣XD
