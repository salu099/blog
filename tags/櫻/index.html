    轉眼間又快到了武陵農場春賞櫻的季節啦~~
    近幾年這兒因賞櫻人氣爆紅
    在總量管制遊客人數前，Salu也有幸前往武陵賞櫻
    雖然不是粉紅佳人大開時前去，但已經覺得很值得囉
    也覺得賞這麼一次就夠了
    反正總量管制，Salu也很清楚自己搶不到這些名額的啦…
    去年年底花了好幾個小時打電話訂房，根本打不進去阿!~
    深深覺得，把這些打電話的精神、體力、時間拿來做其他事情，Salu可能會比較開心點XDDD

    Jul 13, 2013
    2013京阪神奈
    日本 / 京都 / 關西 / 櫻 / 半木之道 / 枝垂櫻

    沿著鴨川往上走
    鴨川上游的賀茂川畔也是種植了整排的櫻花
    在京都府立植物園旁的這一條[半木之道(なからぎのみち)]品種主要是枝垂櫻
    salu每回看京都書籍總會介紹這條無料賞櫻路線
    一直很想前來阿
    這回總算有機會來半木之道看看囉！
    拎著剛在錦市場買來的食物們
    就著河川旁的樹下
    看著櫻花、吹著微微的風
    噢不：天空還不定時的飄著毛毛雨+想分一杯羹而接近我們的鴿子群們
    速速解決手中的食物，還是趕快來賞櫻花吧！

    。2013 日本關西賞櫻去。哲學之道、賞櫻花?! 看人海?!

    Apr 13, 2013
    2013京阪神奈
    日本 / 關西 / 京都 / 哲學之道 / 櫻

    依依不捨的離開白川疏水，繼續往下一站，也就是附近的哲學之道
    99%來到京都賞櫻的人，應該都會到這個地點吧。–哲学の道（てつがくのみち）
    哲學之道以關雪櫻有名。最為人所知的就是哲學家・西田幾多郎常常在這裡散步冥想吧
    去年(2012年/平成24年)salu來京都時，
    打死不願安排此地點，理由很簡單，那就是salu最怕人山人海啦~
    咦，那今年為何安排這裡呢?
    理由也很簡單，因為怕被笑話咩，怕跟人家說來京都賞櫻沒到過哲學之道，被人笑XDD
    這大概就像來到京都沒到過清水寺、金閣寺，會被人說沒來過京都一樣(咦，salu還真沒去過金閣寺也XD)
