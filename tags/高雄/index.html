    台灣第一座天主堂＊高雄玫瑰天主堂

    Aug 14, 2008
    撮る
    高雄 / 台灣南部 / 玫瑰天主堂

    老早以前就想來這拍照啦
    之前不知道他的歷史&名稱, 每回搭車經過都忍不住多看他兩眼
    上網查了一下才發現他可是台灣第一座天主教堂(清朝准許外人來中國傳教後,在台灣蓋的第一座教堂)
    而且還是全台最大的主座教堂(=座堂,就是有主教座位的教堂,通常也被視為教區的中心)
    這座教堂採簡化式哥德式+羅馬式混合風格建築
    拱窗&彩繪玻璃我覺得算是這座教堂最特別&最迷人之處,這座教堂真的超美的也^^

    阿就長的像咩！！總讓我回想起大學的點點滴滴～不過其實兩者還是有很多的相異點啦！（笑）
