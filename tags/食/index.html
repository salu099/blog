    去日本，免不了的就是敗家…
    不過相較於以往，現在Salu比較沒那麼熱衷於在日本購買美妝品了
    一是想要讓行李箱多點空間塞食物
    一是其實很多東西在台灣就買的到
    (當然，那種差價很大又是消耗品的生活必需品除外啦)
    所以今年關西這一趟，主要採買的以零食為主，以及免不了要買些孝敬親朋好友們的伴手禮~

    Jun 20, 2013
    2013京阪神奈
    日本 / 奈良 / 關西 / 食 / 大佛布丁 / 大仏プリン本舗 / 古奈都庵 / 中谷堂 / 艾草麻糬

    既然來到奈良，雖然只有短短半天的時間，美食一定要吃的啦~~~
    奈良有甚麼呢？
    除了人氣排隊美食志津香釜飯，還有烏龍麵麵鬪庵、柿葉壽司、許多大佛相關的產品(如大佛拉麵、大佛燒等)
    不過salu胃袋不夠大，此回上述的品項一個都沒吃到！！釜飯只好等下次囉~~
