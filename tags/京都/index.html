    。2013 日本關西賞櫻去。哲學之道、賞櫻花?! 看人海?!

    Apr 13, 2013
    2013京阪神奈
    日本 / 關西 / 京都 / 哲學之道 / 櫻

    依依不捨的離開白川疏水，繼續往下一站，也就是附近的哲學之道
    99%來到京都賞櫻的人，應該都會到這個地點吧。–哲学の道（てつがくのみち）
    哲學之道以關雪櫻有名。最為人所知的就是哲學家・西田幾多郎常常在這裡散步冥想吧
    去年(2012年/平成24年)salu來京都時，
    打死不願安排此地點，理由很簡單，那就是salu最怕人山人海啦~
    咦，那今年為何安排這裡呢?
    理由也很簡單，因為怕被笑話咩，怕跟人家說來京都賞櫻沒到過哲學之道，被人笑XDD
    這大概就像來到京都沒到過清水寺、金閣寺，會被人說沒來過京都一樣(咦，salu還真沒去過金閣寺也XD)
